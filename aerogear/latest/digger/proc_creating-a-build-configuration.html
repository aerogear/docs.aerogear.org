Creating a Build Configuration

You can use the OpenShift UI to create a new client build for an existing mobile client.

Prerequisites

  • A provisioned Mobile CI/CD service.

  • A Mobile Client

  • To build your mobile client, you need to provide source configuration.

Procedure

  1. Open the OpenShift UI.

  2. Browse to the Project Overview screen and find the mobile client you want to build.

  3. Click on the name of the mobile client to open the details view.

  4. At the top right of the screen, click the drop down and select Create Build.

  5. When the Client Build Form screen opens, give the new build a unique name.
