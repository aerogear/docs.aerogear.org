Deploying a Mobile App

You can use the OpenShift UI to deploy a mobile app by downloading the binary directly to the mobile device. This binary can be downloaded for any successful build in the build history.

Prerequisites

  • A provisioned Mobile CI/CD service.

  • A Mobile Client

  • Created a Build Configuration.

  • Built a Mobile App.

Procedure

  1. Open the OpenShift UI.

  2. In the mobile overview screen for the mobile client, open the Build tab.

  3. Expand the buildconfig that you want to deploy and click Download.

  4. To download the binary to your mobile device, scan the QR code using your mobile device.
