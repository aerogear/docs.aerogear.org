Understanding Analytics

Push collects metadata and payload, and this is accessed using Push Analytics screen.

Push automatically deletes the collected metadata and payload after 30 days by an out-of-hours batch job.

The Analytics screen can been seen in the following image:

Table 1. App Analytics
Title Explanation

Push Messages

The total number of Push Notification Messages sent to Push for dispatching to registered devices.

Notification Dispatched

The total number of Push Notifications dispatched via Push Networks to registered devices.

Notifications Opened

Number of notifications that were delivered to registered device and opened by a user.

Average Open Rate

The result of Push Notifications Opened / Push Notification Dispatched.

Title Explanation

Push Messages

The number of targeted devices for each Push Notification.

Notification Dispatched

The number of opened apps for each Push Notification.
