Sending Push Notifications

The Unified Push admin UI allows you to send a Push Notification to devices.

Prerequisites

[x] Access to the Unified Push admin UI.

Procedure

  1. Open a PushApplication and click on the "Send Notification" button. The Send Push dialog is displayed.

  2. In the Send Push dialog, enter text in the Message text field.

  3. Click the Send Push Notification button.

It is possible to filter the list of receivers, using Alias, Device Types and Category.

Your message is delivered by the 3rd party Push Network to your device.

This is a preview of the https://docs.aerogear.org website.
