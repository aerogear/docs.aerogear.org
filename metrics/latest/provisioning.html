Provisioning the Metrics Service

When provisioning a metrics service, you are prompted to set the following:

  • Grafana Storage Size (Gb)

  • Prometheus Storage Size (Gb)

  • Postgres Storage Size (Gb)

  • Postgres User

  • Postgres Password
